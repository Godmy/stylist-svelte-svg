## 2. Ключевые архитектурные решения (Pivot)

| Было (Гипотеза) | Стало (Реальность) | Обоснование |
| :--- | :--- | :--- |
| `proto → recipe → component` | **`proto → recipe`** | Recipe уже содержит финальный мерж типов. |
| `contract` как UI-фасад | `contract` как системная граница | Contract нужен для Тем и Провайдеров. |
| Сборка через Builder | **Type-Level Fold** | `StructIntersectAll` — это рекурсивная свертка типов. |
| `role` (временный) | **`behavior`** | Более точное описание поведенческих признаков (Traits). |
