## 2. Обновленный словарь соединений (Joints Reference)

| Joint | Статус | Назначение |
| :--- | :--- | :--- |
| **`proto`** | **[LEGACY]** | Временный контейнер. Новые файлы не создавать. |
| **`behavior`** | ✅ Стандарт | Функциональный признак. Что умеет компонент? (напр. `BehaviorClickable`). |

| **`recipe`** | ✅ Стандарт | Основное место для Props компонента. |

## 3. Правила именования и Префиксы

В SAMO 2.0 префиксы — это архитектурные маркеры.

1.  **Поведение**: Префикс **`Behavior`** (напр. `BehaviorClickable`).
2.  **Слоты**: Префикс **`Slot`** (напр. `SlotLabel`).
3.  **Контракты**: Префикс **`Contract`** (напр. `ContractTheme`).
4.  **Рецепты**: Суффикс **`Recipe`** (напр. `ButtonRecipe`).

**Инструкция для ИИ:** При создании новых сущностей в `interface/` использование этих префиксов является обязательным.
